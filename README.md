# Portfolio
The official project for my portfolio.


## Overview
This project is the main source code for my portfolio. It is primarily built with [Next.js](https://nextjs.org/).


## Setup and Installation
Prior to retrieving the source code, ensure you have the following software:
1. Node.js and npm package manager for your machine (see: https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

Next, clone the repository into an empty directory. Then, checkout to the main branch and pull the latest changes. <br>
Run the following commands in a terminal:
```bash
git clone https://github.com/jdinh-782/portfolio.git
git checkout main
git pull
```

Now, install the necessary packages provided with the project using the npm package manager and `package.json` file. Ensure you are at the root of the directory and that it contains this file. <br>
Run the following command in a terminal:
```bash
npm install
```


## Build, Test, and Run
Once the packages are installed, you can begin running the project to view and test changes. Ensure you are at the root of the directory and that it contains the `package.json` file. <br>
Run the following command in a terminal:
```bash
$ npm run start
```
By running the above command, it will build the project and start the app on your local environment. Typically, it can be viewed by navigating to [localhost:4200](http://localhost:4200).
